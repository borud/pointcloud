# Point Cloud Viewer

A simple point cloud viewer built with [Fyne](https://fyne.io/) and OpenGL. Reads PLY files and renders them as interactive 3D point clouds with mouse-controlled rotation and zoom.

![Screenshot](screenshot.png)

## Sample data

The `data/` directory contains sample PLY files from [Artec 3D](https://www.artec3d.com/). These files are provided by Artec Group 3D Scanning Solutions and remain the property of their respective owners.
